# LetsUpgrade JavaScript Assignments

This repository contains JavaScript practice assignments completed as part of the LetsUpgrade JS Essentials learning track.

## Project Structure

```text
LetsUpgrade-JS/
├── README.md
└── Day-2/
	├── index.html
	└── assignment.js
```

## Day 2 Assignment

The `Day-2/assignment.js` file includes solutions for:

1. Searching for a particular character in a string
2. Converting minutes to seconds
3. Searching for an element in an array of strings
4. Displaying array elements containing the character `a`
5. Printing an array in reverse order

## Prerequisites

- [Node.js](https://nodejs.org/) (for running the script in terminal)
- Any modern web browser (for running through `index.html`)

## Run the Assignment

### Option 1: Run with Node.js

From the repository root:

```powershell
Set-Location "C:\LetsUpgrade-JS\Day-2"
node .\assignment.js
```

### Option 2: Run in Browser

Open `Day-2/index.html` in your browser, then open Developer Tools -> Console to view output.

## Verified Output Snapshot

The script was executed successfully and prints all five assignment results, including:

- Character position lookup in `JavaScript`
- `120` minutes converted to `7200` seconds
- Reverse print output: `6 5 4 3 2 1`

## Notes

- Source code uses basic JavaScript concepts: arrays, strings, loops, `forEach`, and `indexOf`.
- Outputs are printed using `console.log`.
